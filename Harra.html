Configuración Inicial - Firebase

Para que el sistema funcione entre múltiples dispositivos, necesitas configurar Firebase (gratuito).

1. Crear cuenta en Firebase
2. Crear proyecto en Firebase Console
3. Activar Firestore Database
4. Copiar configuración aquí
Esta configuración es segura y solo se usa en el navegador

¿Cómo obtener la configuración?

  1. Ve a Firebase Console
  2. Crea un nuevo proyecto o selecciona uno existente
  3. Ve a "Project settings" (engranaje)
  4. En "Your apps", añade una app web
  5. Copia el objeto de configuración
  6. Pégalo en el campo de arriba
